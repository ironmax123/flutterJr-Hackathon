## ホーム画面
- 10分間のタイマー
- カメラ
10分経過すると自動投稿される
<img src='https://github.com/user-attachments/assets/b305be0c-d8fb-4424-bbc5-e417d2843e91' width=30%>

- 自分の情報の更新

- 友達の追加

    - SNSシェア機能
    - 文字入力
